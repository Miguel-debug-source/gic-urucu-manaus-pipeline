# gic-urucu-manaus-pipeline
Modeling framework for geomagnetically induced currents (GICs), pipe-to-soil potential (PSP), and corrosion risk along the Urucu-Manaus equatorial gas pipeline (Brazil). Includes geomagnetic/interplanetary data processing, geoelectric field calculation (1D layered Earth), DSTL pipeline modeling, and corrosion-rate estimation.
